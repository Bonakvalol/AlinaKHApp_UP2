Описание проекта

WPF-приложение "Доска объявлений" для размещения, просмотра и управления объявлениями о продаже товаров и услуг. Система позволяет пользователям создавать объявления, фильтровать их по различным параметрам, завершать сделки и отслеживать финансовые результаты.
Структура проекта
Основные окна приложения:

    MainWindow.xaml - Главное окно со списком объявлений
    LoginWindow.xaml - Окно авторизации пользователей
    RegisterWindow.xaml - Окно регистрации новых пользователей
    ProfitHistoryWindow.xaml - Окно истории прибыли
    CompleteAdWindow.xaml - Окно завершения объявления
    AddAdvertisementWindow.xaml - Окно создания нового объявления

Классы и модели:

    DatabaseHelper.cs - Класс для работы с базой данных
    Advertisement.cs - Модель объявления
    ProfitItem.cs - Модель элемента прибыли

Технологии

    Язык: C#
    Платформа: .NET Framework, WPF
    База данных: SQL Server
    Архитектура: Клиент-серверная

Основные функции

    Авторизация и регистрация пользователей
    Создание и управление объявлениями
    Фильтрация объявлений по городу, категории, типу и статусу
    Поиск по названию и описанию
    Завершение сделок с расчетом прибыли
    Просмотр истории финансовых операций

База данных

Проект использует реляционную базу данных SQL Server со следующими основными таблицами:

    Users (пользователи)
    Advertisements (объявления)
    Categories (категории)
    Cities (города)
    AdTypes (типы объявлений)
    AdStatuses (статусы объявлений)
